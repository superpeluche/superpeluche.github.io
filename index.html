L'Agence

Créée en 2014, l'agence PandaBox est le fruit de la rencontre imaginaire entre deux spécialistes du web qui avaient en commun une vision tournée vers l'expérience ainsi qu'un amour inconditionnel pour les pandas. Chez nous les pandas ne savent pas que ronger des bambous !

Intégration

Nous mettons nos talents en oeuvre pour délivrer un code propre et parfaitement sémantique, pour vous aider dans votre référencement.

Sites Responsive

Nous sommes conscients de l'enjeu d'acquérir les utilisateurs mobiles. C'est pourquoi nos sites fonctionnent sur tout types d'appareils.

Web Design

Nos pandas designers se démènent pour délivrer les meilleurs designs possible afin de rendre l'éxpérience agréable du début à la fin.

Stratégie Digitale

Nous vous accompagnons dans l'élaboration de votre stratégie afin de vous assurer une présence sur le web à long terme.

services

Notre agence offre une large palette de services pour répondre à tout les besoins possibles en matière de présence sur le web. Notre expertise nous permet de réaliser presque tout types de projets.

Développement Front-End

Vous avez déja une maquette ou un design sur lequel travailler ? Nous avons le code pour lui donner vie. HTML5, CSS3, jQuery ... Rien ne nous fait peur !

Developpement Back-End

Vous avez un projet plus ambitieux qui nécessite la création et la mise en ligne d'un CMS open source ? Nos solutions personnalisées sont la réponse qu'il vous faut.

SEO

Avoir un site interet ne suffit pas, il faut pouvoir le rendre visible. Notre équipe met toutes les bonnes pratiques en place pour vous faire accéder en premiere page de google.

Social Media

Nous vous aidons à créer et gérer vos communautés en ligne pour asseoir votre présence sur le web et créer de l'intéraction.

Nos Créations

E-commerce, sites vitrines, portfolios ... Nous travaillons pour tout types de clients sur tout types de projets. Voici un aperçu de ce que nos pandas ont réalisé ces derniers mois.
